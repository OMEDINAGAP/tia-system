Proceso de Tarjeta de Identificación Aeroportuaria

En este portal podrás completar los pasos necesarios para la obtención de tu credencial aeroportuaria.

📄 Requisitos del trámite

Aquí encontrarás toda la documentación necesaria para solicitar la TIA, incluyendo identificación oficial, carta laboral, antecedentes no penales, entre otros.

📤 Envío de documentación

En esta sección podrás cargar tus documentos para validación. Solo el personal autorizado podrá realizar este proceso.

🎥 Curso de seguridad obligatorio

Antes de obtener tu credencial, deberás completar el curso de seguridad aeroportuaria. El sistema registrará tu progreso y validará tu participación.
